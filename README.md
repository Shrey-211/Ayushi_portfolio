## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org) v18+
- [Bun](https://bun.sh) (recommended) or npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Ayushimishra054/Portfolio.git
cd Portfolio

# Install dependencies
bun install
# or
npm install
```

### Development

```bash
bun run dev
# or
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the portfolio locally.

### Build for Production

```bash
bun run build
# or
npm run build
```

### Preview Production Build

```bash
bun run preview
# or
npm run preview
```

## 🎨 Design Details

- **Typography**: Cormorant Garamond (serif, headlines) + Jost (sans-serif, body)
- **Color Palette**: Deep navy-to-plum gradient background with blush pink (`#F5C8D6`) and sky blue (`#8CC3F8`) accents
- **Effects**: Film grain overlay, scroll-driven reveals, glassmorphism cards, custom cursor with mix-blend-mode
- **Animations**: CSS keyframe animations, cubic-bezier easing, perspective 3D door transitions
